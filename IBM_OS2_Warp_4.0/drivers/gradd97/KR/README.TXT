  발췌
  __________

  GRADD Graphics BBS 드라이버 설치 디렉토리를 설정하려면, 다음을 수행하십시오.

  1.  OS/2 전체 화면이나 OS/2 창 세션을 여십시오.

  2.  GRADDBB*.EXE 자체 발췌 zip 파일(정확한 이름은 릴리스 빌드 번호에 좌우됩니다)을
      실행하기 전에, GRADD Graphics BBS 드라이버 설치 파일을 저장하려는 "설치
      디렉토리"로 현재 디렉토리를 변경하십시오. C:\BBS\GRADD를 제안합니다.
      OS/2 명령 프롬프트에서, 다음 명령을 입력하십시오.

      C:
      MD C:\BBS
      MD C:\BBS\GRADD
      CD C:\BBS\GRADD

  3.  "-DIR" 및 "-OVER" 옵션을 GRADDBB*.EXE 자체 발췌 zip 파일에서
      사용하여 설치 디렉토리 트리를 작성하십시오. 먼저 GRADDBB*.EXE 자체
      발췌 zip 파일을 설치 디렉토리로 복사하려고 할 수 있습니다.
      그런 다음 OS/2 명령 프롬프트에서, 다음 명령을 입력하십시오.

      GRADDBB* -DIR -OVER

  4.  SET LANG=을 사용하여 올바른 언어를 설정하십시오. 언어가 이미
      CONFIG.SYS에 제대로 설정되어 있을 수 있습니다.  그렇지 않은 경우, OS/2
      명령 프롬프트에서, 이 SET LANG= 명령 중 하나를 입력하여 사용자 언어를 선택할 수 있습니다.

      SET LANG=ar_AA    아랍(아랍어 그래픽 형식화가 제공된 영어)
      SET LANG=pt_BR    브라질
      SET LANG=zh_CN    간체 한자
      SET LANG=de_DE    독일
      SET LANG=da_DK    덴마크
      SET LANG=es_ES    스페인
      SET LANG=el_GR    그리스(그리스어 그래픽 형식화가 제공된 영어)
      SET LANG=fi_FI    핀란드
      SET LANG=fr_FR    프랑스
      SET LANG=iw_IL    이스라엘(헤브라이어 그래픽 형식화가 제공된 영어)
      SET LANG=it_IT    이탈리아
      SET LANG=jp_JP    일본
      SET LANG=ko_KR    한국
      SET LANG=nl_NL    네덜란드
      SET LANG=no_NO    노르웨이
      SET LANG=sv_SE    스위스
      SET LANG=zh_TW    타이완
      SET LANG=en_US    영어(기본값)

      각 언어는 이 README.TXT 파일의 고유한 사본을 가지며, 영어는 물론
      기타 언어 종속 파일에서 변환될 수 있습니다.



                OS/2 GRADD 디바이스 드라이버                                페이지 2

      정보를 설치 디렉토리에서 사용자 부트 드라이브로 설치하는 데
      사용될 수 있습니다.
      임의의 옵션 없이 SETUP을 실행하는 것은 선택 또는 기본 언어에서

      README 파일에서 자세힌 설명됩니다). 이 도움말을 보려면,
      OS/2 명령 프롬프트에서, 다음 명령을 입력하십시오.

      SETUP

  GRADD(GRAPHICS ADAPTER DEVICE DRIVERS)
  _______________________________________

  GRADD는 새 그래픽 하드웨어를 사용할 수 있게 되면 더 용이하게 지원하게
  하는 구조입니다.  확장된 PM 어플리케이션 성능은 가속화된
  GRADD 드라이버를 사용할 때 인식할 수 있습니다.

  새로 갱신된 여러 GRADD 드라이버가 이 패키지와 함께 포함됩니다.


  지원되는 기능
  __________________

  GRADD 디바이스 드라이버는 가속화 및 비가속화된 디스플레이 그래픽과
  확장된 컬러 농도와 해상도의 완벽한 지원을 제공합니다.
  이 드라이버는 OS/2 32비트 플랫 메모리 모델에 따르며
  OS/2 32비트 그래픽 엔진에서 32비트 Presentation Manager* 그래픽
  디스플레이 드라이버로서 기능하도록 디자인되었습니다.


  포함된 드라이버
  ________________

  가속화된 GRADD 드라이버는 다음과 같습니다.

      * ATI Mach64/Rage GRADD           (M64GRADD)
      * Chips & Technology 6555X GRADD  (CHPGRADD)
      * Matrox Millenium/Mystique GRADD (MGAGRADD)
      * NeoMagic GRADD                  (NMGRADD)
      * S3 86x/96x/Trio GRADD           (S3GRADD)
      * S3 Virge/Trio3D GRADD           (S3DGRADD)

  또한 범용 VGA 및 SVGA 지원도 포함됩니다.

      * Video Graphics Array GRADD      (VGAGRADD)
      * Generic SVGA GRADD              (GENGRADD)

  가속화된 GRADD 또는 PM 드라이버가 사용자 어댑터용으로 포함되지 않은 경우,
  GENGRADD는 확장된 해상도 및 컬러 농도로 디스플레이 그래픽에 대한
  UNACCELERATED 지원을 제공할 수 있습니다.  GENGRADD는
  VESA compliant BIOS를 포함하는 대부분의 어댑터로 작업하며,
  시스템 구성에 따라 다양한 성능은 가속화된 드라이버로서 좋지 않습니다.

  지원되는 해상도
  _____________________

  지원되는 해상도 및 컬러 농도는 어댑터에 종속되며 그래픽 메모리의 양과
  어댑터 구현에 따라 다양합니다.


  호환성
  _____________

  이 그래픽 디바이스 드라이버는 현재 다음과 호환가능합니다.

        fixpak 35 이상인 모든 OS/2 3.0 버전
        fixpak 레벨 5 이상인 모든 OS/2 4.0 버전
        전자 업무용 OS/2 Warp Server의 모든 버전

  SETUP 설치 명령은 수용할 수 없는 OS/2 버전이나
  fixpak 레벨에서의 설치를 거부합니다.

  또한 이 그래픽 드라이버를 설치한 후에, 나중에 Fixpak 5(OS/2 4.0)
  또는 Fixpak 35(OS/2 3.0) 이전에 필요한 모든 Fixpak 레벨을 설치해제하려는 경우
  우선 사용자 컴퓨터를 VGA로 복원하여 GRADD BBS 드라이버를 제거해야 합니다.

  영어 메시지를 사용하지 않고, 위의 "발췌" 섹션에 있는
  SET LANG= 목록에 없는 SBCS 언어용으로 설치된 SBCS 드라이버는
  선택된 언어용으로 제대로 기능하지 않습니다.

  기능적인 제한사항
  _______________________

  현재 알려진 제한사항은 다음과 같습니다.

  1) IBM은 나열된 가속화된 드라이버의 그래픽 디바이스와 통합하여 여러
     컴퓨터와 어댑터를 테스트했습니다.  PCI 버스는 특정 요구사항이 아니지만
     테스트는 PCI 버스 그래픽이 있는 컴퓨터에서만 수행되었습니다.

  2) ISA 버스 그래픽 어댑터에서 Sniff 테스트를 수행했습니다. 그러나
     이전 "PM" 드라이버를 복원하려고 시도할 때, 때때로 어댑터를 검출하는
     문제점이 있었습니다. VGA를 복원한 후에 어댑터를 초기화하기 위해
     컴퓨터의 전원을 끄면 문제점이 해결되어야 합니다.

     ATI Mach 64 ISA Bus 어댑터는 현재 지원되지 않습니다.

  3) VGAGRADD를 제외한 모든 GRADD 드라이버는 VESA** compliant 비디오
     BIOS(버전 1.2 이상)나 해당 SVGADATA.PMI 파일을 요구합니다. IBM에서
     SVGA PM 드라이버를 지원하는 비 VESA 어댑터의 경우, OS/2 또는 DOS 전체
     화면 프롬프트에서 "SVGA ON INIT" 명령과 함께 다음과 같은
     GRADD Graphics BBS 드라이버 SETUP 설치 명령으로 설정된 대체 SGVA 드라이버로서
     자주 사용될 수 있습니다.

  4) M64GRADD 및 S3GRADD: 하드웨어 커서를 지원하는 특정 어댑터에서 문제점이
     발생했습니다.  솔리드 검은색 사각형으로 표시되는 포인터로부터 커서가
     패스되는 화면에서 발생하는 손상에 이르기까지 증상이 다양합니다.

        SET HW_CURSOR=OFF

  디스켓으로 복사
  ____________________


  설치 디렉토리 트리로 발췌하고 나면,
  OS/2 BACKUP 명령을 사용하여 백업 또는 전송 목적으로 디스켓에 GRADD 드라이버
  설치 파일을 위치시킬 수 있습니다.

  1. OS/2 전체 화면이나 OS/2 창 세션을 여십시오.

  2. "BACKUP C:\BBS\GRADD A: /S"를 입력하고 Enter 키를 누르십시오.

  3. 필요한 만큼 사전 포맷된 여러 디스켓을 레이블하여 넣으십시오.

  4. BACKUP 명령에 대한 자세한 정보는 HELP BACKUP을 사용할 수 있습니다.

  동일한 디스켓을 같거나 다른 컴퓨터로 복원하려면 다음을 수행하십시오.

  1. OS/2 전체 화면이나 OS/2 창 세션을 여십시오.

  2. "RESTORE A: C:\BBS\GRADD /N /S"를 입력하고 Enter 키를 누르십시오.

  3. 필요하면 이전에 백업된 디스켓을 넣으십시오.

  4. RESTORE 명령에 대한 자세한 정보는 HELP RESTORE를 사용할 수 있습니다.

  IBM은 디스켓에서 직접 이 드라이버를 설치하도록 제공하지 않습니다.

  사용자 디스플레이를 VGA 모드로 설정
  ________________________________

  사용자 디스플레이가 동기 상태이며 그렇지 않으면 사용가능한 데스크탑을
  가진 경우, 선택 설치를 사용하여 사용자 디스플레이를 VGA 모드로 설정하도록
  권장합니다.
  그렇지 않으면, 다음과 같은 절차를 사용하여 VGA 모드로 리턴하십시오.

      1.  사용자 컴퓨터를 켜십시오.  컴퓨터가 이미 켜져 있으면 시스템 종료하거나
          Ctrl+Alt+Del를 눌러 재시작하십시오.
      2.  화면의 왼쪽 상단 구석에 작은 흰색 상자가 나타나면,
          Alt+F1을 누르십시오.
      3.  복구 선택 화면이 나타나면, 운영 시스템 버전에 따라 기능 키 F3이나
          "V" 문자를 누르십시오.
      4.  선택 설치를 실행하여 Windows 드라이버를 변경하십시오.

  컴퓨터가 부트되면, 사용자 디스플레이 어댑터가 복원되어
  VGA 모드에서 작업할 수 있습니다. 새 디스플레이 어댑터나 그래픽 디바이스 드라이버를
  설치하려는 경우, OS/2 Warp에 대한 사용자 안내서에서 "비디오 프로시듀어" 장을
  참조하십시오.


  무인 설치
  _____________________

  자체 발췌 zip 파일 설치 디렉토리에서 GRADD 드라이버의 무인 설치를 수행하려면,
  다음을 수행하십시오.

  1.  OS/2 전체 화면이나 OS/2 창 세션을 여십시오.

  2.  발췌된 SETUP.CMD 파일을 실행하기 전에, SET LANG=을 사용하여 올바른
      언어를 설정하십시오. 언어가 이미
      CONFIG.SYS에 제대로 설정되어 있을 수 있습니다.  이 README의 "발췌" 섹션에서는
      예제가 들어 있습니다.



                OS/2 GRADD 디바이스 드라이버                                페이지 8

  3.  OS/2 명령 프롬프트에서, 다음 명령을 입력하여 필요한 모든 파일을
      OS/2 부트 드라이브로 복사하십시오.

          <SrcPath>SETUP xxx  <SrcPath>   <BootDrive>   /u 

      여기서
          xxx는 다음 중 하나입니다.

              "ATI"   - ATI Mach64/Rage Family GRADD
              "CHP"   - Chips & Technology 6555x Family GRADD
              "DYN"   - Dynamically Selected GRADD
              "GEN"   - Generic VESA Unaccelerated GRADD
              "MGA"   - Matrox Millenium & Mystique Family GRADD
              "NM"    - NeoMagic 2090/2093/2097/2160/2200/2360 Family GRADD
              "S3"    - S3 86x/96x/Trio Family GRADD
              "VGA"   - Video Graphics Array (VGA) GRADD
              "VIRGE" - S3 Virge/Trio3D Family GRADD

          SETUP의 <SrcPath>는 SETUP이 현재 디렉토리에 없는 경우 필요합니다.

           <SrcPath>  매개변수는 선택적이며(다음 매개변수가 지정된 경우 필요)
                      설치 디렉토리입니다.

           <BootDrive>는 선택적이며 OS/2가 설치된 드라이브입니다.

           /u는 선택적이며 CID 설치에서와 같이 무인 설치를 나타냅니다.
               CID 응답 파일은 필요하지 않습니다.

      그런 다음 Enter를 누르십시오.

  4.  그렇게 수행하도록 프롬프트되면, 시스템 종료한 후 컴퓨터를 재시작하십시오.

      주:

      1) 이 드라이버의 설치 중, DISPLAY.LOG 및 DSPINSTL.LOG
         파일은 OS2\INSTALL 디렉토리에서 작성되거나 추가되었습니다.  이 파일은
         갱신된 OS/2 시스템 파일을 식별하며 설치가 성공적인지 여부와 실패 원인을
         나타냅니다.  이 파일의 내용은 설치 문제점을 IBM에 보고할 필요가 있는
         경우 유용합니다.

      2) 목표 파일이 소스 파일보다 더 새로운 특정 파일을 중첩하려는 지 여부에
         대해 프롬프트될 수 있습니다. 일반적으로, 이전에 드라이버에서 특정
         문제점을 경험한 경우를 제외하고는 YES로 응답하고 재설치합니다. 








         OS/2 GRADD 디바이스 드라이버                                페이지 9

  5.  컴퓨터를 재부트한 후에, 다음을 수행하여 현재 디스플레이를 제대로 구성하고
      적절한 해상도와 화면갱신 비율을 선택하십시오.

      a.  OS/2 시스템 폴더를 여십시오.
      b.  시스템 설정 폴더를 여십시오.
      c.  시스템 객체를 여십시오.
      d.  설정 노트북이 나타나면, 화면 탭을 선택하십시오.
          그런 다음 화면 페이지 2로 가십시오.
      e.  VESA** Display Data Channel (DDC) 1 및 2B 프로토콜이 자동으로
          DDC 디스플레이를 감지하도록 지원됩니다.
          디스플레이가 DDC 호환가능하면, 화면 페이지 2가 사용자 디스플레이의
          설명을 표시해야 합니다.
          디스플레이가 DDC 호환가능하지 않으면,
          화면 페이지 2의 디스플레이 목록에서 사용자 디스플레이를 선택하십시오.
      f.  화면 페이지 1로 리턴하여 해상도와 화면갱신 비율을 선택하십시오.
          어댑터와 디스플레이에 따라, 화면갱신 비율은 선택가능하지 않을 수 있습니다.
      g.  설정 노트북을 닫으십시오.
      h.  컴퓨터를 시스템 종료하고 재시작하십시오.

  CID 환경에서 설치
  _______________________________

  주:

  1.  서버에서 디렉토리(예를 들어 X:\BBS\GRADD)를 작성하고 "발체" 섹션에서
      설명한 대로 자체 발췌 GRADDBB*.EXE 파일에서 X:\BBS\GRADD 설치 디렉토리로
      설치 파일을 발췌하십시오.

  2.  CID(Configuration Installation Distribution) 메소드를 사용하여 클라이언트에
      정상적으로 OS/2를 설치해야 합니다.

  3.  디스플레이 선택, 해상도 및 화면갱신 비율을 구성하려면, 이 패키지에서
      "README.CFG" 파일을 참조하십시오.

  CID를 사용하여 디바이스 드라이버를 설치하려면, 다음 정보를 사용하여
  LCU 명령 파일을 수정하십시오.  아래 예제는 ATI Mach64/Rage GRADD
  (M64GRADD)를 설치합니다.

  주:  다음 정보는 지침으로서 사용됩니다.
         LCU 명령 파일은 다를 수 있습니다.

    /*****************************************************/

    NUM_INSTALL_PROGS = 15
