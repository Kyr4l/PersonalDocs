  解壓縮
  __________

  要設定 GRADD 圖形 BBS 驅動程式安裝目錄，請執行下列步驟：

  1.  開啟一個 OS/2 全螢幕或 OS/2 視窗階段作業。

  2.  在執行 GRADDBB*.EXE 自行解壓縮 zip 檔之前（實際名稱
      會根據版次建置號碼而不同），請將現行目錄變更為
      您要儲存 GRADD 圖形 BBS 驅動程式安裝檔的
      「安裝目錄」。建議您使用 C:\BBS\GRADD。
      在 OS/2 指令提示下，鍵入下列指令：

      C:
      MD C:\BBS
      MD C:\BBS\GRADD
      CD C:\BBS\GRADD

  3.  在 GRADDBB*.EXE 自行解壓縮 zip 檔案使用 "-DIR" 和 "-OVER" 選項，
      以建立安裝目錄樹狀結構。您可以先將 GRADDBB*.EXE 自行解壓縮檔案複製到
      安裝目錄。然後在 OS/2 指令提示下，鍵入下列指令：

      GRADDBB* -DIR -OVER

  4.  使用 SET LANG= 來設定正確的語言。在您的 CONFIG.SYS 中可能已經正確
      設定好語言。如果沒有，請在 OS/2 指令提示下，
      鍵入以下其中一個 SET LANG= 指令來選擇您的語言：

      SET LANG=ar_AA    阿拉伯文（採用阿拉伯文圖形格式的英文）
      SET LANG=pt_BR    巴西文
      SET LANG=zh_CN    簡體中文
      SET LANG=de_DE    德文
      SET LANG=da_DK    丹麥文
      SET LANG=es_ES    西班牙文
      SET LANG=el_GR    希臘文（採用希臘文圖形格式的英文）
      SET LANG=fi_FI    芬蘭文
      SET LANG=fr_FR    法文
      SET LANG=iw_IL    以色列文（採用希伯來文圖形格式的英文）
      SET LANG=it_IT    義大利文
      SET LANG=jp_JP    日文
      SET LANG=ko_KR    韓文
      SET LANG=nl_NL    荷蘭文
      SET LANG=no_NO    挪威文
      SET LANG=sv_SE    瑞士
      SET LANG=zh_TW    台灣
      SET LANG=en_US    英文（預設值）

      每一種語言都有自己的 README.TXT 檔，檔案可能是從英文翻譯過來，
      並具有其他依語言版本而定的檔案。

                OS/2 GRADD 裝置驅動程式                                第 4 頁

  圖形配接卡裝置驅動程式（GRADD）
  _______________________________________

  GRADD 是一種更容易支援新圖形硬體的架構。
  在使用加速的 GRADD 驅動程式時，
  就會發現加強型 PM 應用程式的效能。

  這個套裝軟體包含了數種新增及更新的 GRADD 驅動程式。


  支援的特性
  __________________

  GRADD 裝置驅動程式為加速和未加速顯示器圖形提供廣泛的支援，
  以及進階的色彩深度與解析度。
  這些支援符合 OS/2 32 位元平面記憶體模型，
  其設計是為了在 OS/2 32 位元圖形引擎下作為
  32 位元「呈現管理程式」* 圖形顯示器驅動程式使用。


  內含的驅動程式
  ________________

  這份套裝軟體包含以下的加速 GRADD 驅動程式：

      * ATI Mach64/Rage GRADD           (M64GRADD)
      * Chips & Technology 6555X GRADD  (CHPGRADD)
      * Matrox Millenium/Mystique GRADD (MGAGRADD)
      * NeoMagic GRADD                  (NMGRADD)
      * S3 86x/96x/Trio GRADD           (S3GRADD)
      * S3 Virge/Trio3D GRADD           (S3DGRADD)

  同時也包含了同屬 VGA 及 SVGA 的支援。

      * 影像圖形陣列 GRADD           (VGAGRADD)
      * 通用 SVGA GRADD              (GENGRADD)

  如果未包含您的配接卡專用的加速 GRADD 或 PM 驅動程式，
  GENGRADD 可以為顯示器圖形提供「未加速」的支援，
  並提供加強型的解析度和色彩深度。GENGRADD 可以和大部分包含 VESA 相容 BIOS 的配接卡一起使用；
  其效能會根據系統架構而有所不同，但無法和加速的驅動程式一樣好。

  _____________

        所有的 OS/2 3.0 版本（具 FixPak 層次 35 或更新的版本）
        所有的 OS/2 4.0 版本（具 FixPak 層次 5 或更新的版本）
        所有的 OS/2 Warp Server for e-business 版本

  同時，在安裝這些圖形驅動程式後，如果您想要再將任何必要的 FixPak 解除安裝到
  FixPak 5（OS/2 4.0）或 FixPak 35（OS/2 3.0）以前的 Fixpak 層次，
  您應該先將電腦還原成 VGA，然後再移除 GRADD BBS 驅動程式。

  也能為選擇的語言正確運作。

  功能限制
  _______________________

  目前已知的限制包括下列各項：

  1) IBM 已經測試過許多電腦和配接卡，並整合列出的加速驅動程式
     所用的圖形裝置。這些測試只在使用 PCI 匯流排圖形的電腦上執行，
     不過 PCI 匯流排並非特定的需求。

  2) ISA 匯流排配接卡已通過 Sniff 測試。不過在嘗試
     還原先前的 "PM" 驅動程式時，有時候會無法偵測
     配接卡。在還原 VGA 之後，關閉電腦的電源來起始設定配接卡，
     應該可以解決問題。

     目前並未支援 ATI Mach 64 ISA 匯流排配接卡。

  3) 所有的 GRADD 驅動程式（除了 VGAGRADD 之外），都需要 VESA**相容的影像
     BIOS（1.2 版或更新的版本）或適當的 SVGADATA.PMI 檔。對於較舊的非 VESA
     配接卡，如果已經有 IBM 所提供的 SVGA PM 驅動程式支援， GENGRADD 通常
     可以作為替代的 SVGA 驅動程式，方法是在重新開機以使用 GRADD 驅動程式前，
     從 OS/2 或 DOS 全螢幕提示下，使用 GRADD 圖形 BBS 驅動程式 SETUP 安裝指令
     以及 "SVGA ON INIT" 指令來建立 SVGADATA.PMI 檔。


                OS/2 GRADD 裝置驅動程式                                第 6 頁

  4) M64GRADD 及 S3GRADD：某些具備硬體游標支援的配接卡
     有問題。徵兆包括指標變成實心黑色矩形以及游標移動
     時螢幕畫面不良。我們已經提供一個強制使用軟體游標
     的替代方案。請將以下這一行新增至 CONFIG.SYS，
     然後重新開機，就能解決這個問題：

        SET HW_CURSOR=OFF

  您可以經由網際網路來尋找 GRADD 和其他圖形裝置驅動程式的最新修訂程式及更新，網址是：

     http://service5.boulder.ibm.com/pspfixpk.nsf

  在上述的 URL 中，選擇 "All Fix packages - By Product"，然後搜尋 "GRADD"。


  複製到磁片
  ____________________


  一旦解壓縮到安裝目錄樹狀結構，您就可以將 GRADD 驅動程式安裝檔案
  放置到磁片上，以便利用 OS/2 BACKUP 指令來進行備份或轉送：

  1. 開啟一個 OS/2 全螢幕或 OS/2 視窗階段作業。

  2. 鍵入 "BACKUP C:\BBS\GRADD A: /S" 並按下 Enter 鍵。

  3. 標示並插入所需的預先格式化磁片。

  4. 您可以使用 HELP BACKUP 來取得關於 BACKUP 指令的相關資訊。

  如果要將相同的磁片回存到相同或不同的電腦上：

  1. 開啟一個 OS/2 全螢幕或 OS/2 視窗階段作業。

  2. 鍵入 "RESTORE A: C:\BBS\GRADD /N /S" 並按下 Enter 鍵。

  3. 視需要插入先前備份的磁片。

  4. 您可以使用 HELP RESTORE 來取得關於 RESTORE 指令的相關資訊。

  IBM 並未提供直接從磁片安裝這些驅動程式的服務。

  安裝
  ____________

  初步的步驟
  _________________

  以下的步驟可讓您的電腦做好安裝 GRADD 裝置驅動程式的準備：

  1.  您應該關閉其他的應用程式，特別是 WIN-OS/2 應用程式，
      使得裝置驅動程式安裝可以取代已開啟的應用程式可能
      鎖定成唯讀的裝置驅動程式檔案。

  2.  如果您之前安裝了 SVGA 驅動程式，請使用本 README 的
     「將顯示器設定為 VGA 模式」一節中的程序，將解析度回復成 VGA。


  將顯示器設定為 VGA 模式
  ________________________________

  如果您的顯示器已經同步化，或者您有可以使用的「桌面」，
  建議您使用「選擇性安裝」來將顯示器設定為 VGA 模式。
  否則，請使用以下的程序來回復成 VGA 模式。

      1.  開啟您的電腦。如果您的電腦已經開啟，請關機或按 Ctrl+Alt+Del 來重新啟動。
      2.  當小型白色方塊出現在螢幕左上角時，請按 Alt+F1。
      3.  當「恢復選項」螢幕出現時，按功能鍵 F3 或 字母 "V"（視作業系統版本而定）。
      4.  執行「選擇性安裝」來變更 Windows 驅動程式。

  一旦電腦啟動後，您的顯示卡會還原，並且可以在 VGA 模式中運作。
  如果您要安裝新的顯示卡或圖形裝置驅動程式，
  請參閱「OS/2 Warp 使用手冊」中的「影像程序」一章。


  手動式安裝作業
  _____________________

  如果要從自行解壓縮的 zip 檔案安裝目錄中，
  執行 GRADD 驅動程式的手動安裝，
  請執行以下步驟：

  1.  開啟一個 OS/2 全螢幕或 OS/2 視窗階段作業。

  2.  在執行已解壓縮的 SETUP.CMD 檔案之前，請使用 SET LANG= 來設定正確的語言。
      您的 CONFIG.SYS 中可能已經正確設定好語言。本 README 檔中的 "解壓縮" 一節有提供許多範例。



                OS/2 GRADD 裝置驅動程式                                第 8 頁

  3.  在 OS/2 指令提示下，鍵入下列指令來將所有必要的檔案複製到 OS/2 啟動磁碟機：

          <SrcPath>SETUP xxx  <SrcPath>   <BootDrive>   /u 

      其中：
          xxx 是以下其中一項：

              "ATI"   - ATI Mach64/Rage 系列 GRADD
              "CHP"   - Chips & Technology 6555x 系列 GRADD
              "DYN"   - 動態選取的 GRADD
              "GEN"   - 同屬 VESA 未加速 GRADD
              "MGA"   - Matrox Millenium & Mystique 系列 GRADD
              "NM"    - NeoMagic 2090/2093/2097/2160/2200/2360 系列 GRADD
              "S3"    - S3 86x/96x/Trio 系列 GRADD
              "VGA"   - 影像圖形陣列 (VGA) GRADD
              "VIRGE" - S3 Virge/Trio3D 系列 GRADD

          如果 SETUP 不在現行目錄中，則需要 SETUP 的 <SrcPath>。

           <SrcPath>  參數是選用的（如果有指定下一個參數則是必要的）而且是安裝目錄。

           <BootDrive>  是選用的，而且是安裝 OS/2 的磁碟機。

           /u  是選用的，可指定無人式安裝，如 CID 安裝。CID 回應檔是不必要的。

      然後按 Enter 鍵。

  4.  提示關機時，請照做，然後重新啟動電腦。
