  概述
  __________

  要设置 GRADD 图形 BBS 驱动程序安装目录，请完成以下操作：

  1.  打开 OS/2 全屏幕或者 OS/2 窗口阶段作业。

  2.  在运行 GRADDBB*.EXE 自解压 zip 文件前 ( 文件名称是根据发行版的构建号码决定的 )，
      将当前目录切换到希望保存 GRADD 图形驱动程序安装文件的“安装目录”。我们建议使用
      C:\BBS\GRADD。在 OS/2 命令提示符下，输入以下命令：

      C:
      MD C:\BBS
      MD C:\BBS\GRADD
      CD C:\BBS\GRADD

  3.  为了创建安装目录树，在使用 GRADDBB*.EXE 自解压文件时选择“-DIR”
      和“-OVER”选项。首先，您可以将 GRADDBB*.EXE 自解压文件复制到安
      装目录。然后在 OS/2 命令提示符下，输入以下命令：

      GRADDBB* -DIR -OVER

  4.  使用 SET LANG= 设置正确的语言。在 CONFIG.SYS 中已经可能正确的设置了语言。
      如果没有，在 OS/2 命令提示下，输入以下的 SET LANG= 命令来选择您的语言。

      SET LANG=ar_AA    对应于阿拉伯语 ( 使用了阿拉伯图形格式的英语 )
      SET LANG=pt_BR    对应于巴西
      SET LANG=zh_CN    对应于简体中文
      SET LANG=de_DE    对应于德国
      SET LANG=da_DK    对应于丹麦
      SET LANG=es_ES    对应于西班牙
      SET LANG=el_GR    对应于希腊 ( 使用了希腊图形格式的英语 )
      SET LANG=fi_FI    对应于芬兰
      SET LANG=fr_FR    对应于法国
      SET LANG=iw_IL    对应于以色列 ( 使用了希伯莱图形格式的英语 )
      SET LANG=it_IT    对应于意大利
      SET LANG=jp_JP    对应于日本
      SET LANG=ko_KR    对应于韩国
      SET LANG=nl_NL    对应于荷兰
      SET LANG=no_NO    对应于挪威
      SET LANG=sv_SE    对应于瑞士
      SET LANG=zh_TW    对应于中国台湾
      SET LANG=en_US    对应于英语 ( 缺省值 )

      每种语言具有一个对应此 README.TXT 文件的自己的副本，它可能与其他特定于语言的
      文件一样，是从英语文件翻译过来的。

                OS/2 GRADD 设备驱动程序                                第四页

  图形适配器设备驱动程序 (GRADD)
  ______________________________

  GRADD 是一种硬件结构，当出现新的图形硬件时它能更容易地对新图形硬件提供支持。当使用
  GRADD 加速驱动程序时可以感觉到 PM 应用程序性能的提升。 

  在本包中包含了一些新的更新过的 GRADD 驱动程序。


  支持的功能
  __________

  GRADD 设备驱动程序对加速和未加速的图形显示以及高分辨率和色彩深度提供了无缝的支持。
  它们符合 OS/2 32 位平面存储模型，并且它们被设计为可以在 OS/2 32 位图形引擎下作为
  32 位显示管理器图形显示设备驱动程序工作。


  包含的驱动程序
  ______________

  包含了下列 GRADD 加速驱动程序：

      * ATI Mach64/Rage GRADD           (M64GRADD)
      * Chips & Technology 6555X GRADD  (CHPGRADD)
      * Matrox Millenium/Mystique GRADD (MGAGRADD)
      * NeoMagic GRADD                  (NMGRADD)
      * S3 86x/96x/Trio GRADD           (S3GRADD)
      * S3 Virge/Trio3D GRADD           (S3DGRADD)

  同样包含了对标准 VGA 和 SVGA 的支持。

      * 视频图示阵列 GRADD      (VGAGRADD)
      * 标准 SVGA GRADD              (GENGRADD)

  如果不包含您适配器的 GRADD 或 PM 加速驱动程序，那么 GENGRADD 可能会以
  增强分辨率和色彩深度为显示图形提供未加速的支持。GENGRADD 可以与大多数
  的适配器包括 VESA BIOS 一起工作；而且性能（根据系统配置会有改变）将不
  如加速驱动程序。 











                OS/2 GRADD 设备驱动程序                                第五页

  支持的分辨率
  ____________

  所支持的分辨率和色彩深度是和适配器相关的，并且会根据图形存储器的容量和适配器
  的具体情况有所改变。 


  兼容性
  ______

  当前，这些图形设备驱动程序与下列版本兼容：

        带有 fixpak 级别 35 或更新版本的 OS/2 3.0 的所有版本
        带有 fixpak 级别 5 或更新版本的 OS/2 4.0 的所有版本
        OS/2 Warp Server for e-business 的所有版本

  当在不符要求的 OS/2 版本或 fispak 级别上进行安装时，SETUP 安装命令会拒绝
  安装。

  同样在安装这些图形驱动程序后，如果您想解除安装所要求的 Fixpak 级别，
  您应该将您的计算机恢复到 VGA 方式以除去 GRADD BBS 驱动程序。

  除了英语信息以外，为了未列在 SET LANG= 列表中的 SBCS 语言而安装的 SBCS  驱动程序，
  它们也可以对应所选择的语言正常工作，该列表在前面的名为“解压”的章节中。

  功能上的局限
  ____________

  当前已知的局限包括：

  1) 对于列出的加速驱动程序，IBM 已经结合图形设备测试了很多计算机和适配器。
     尽管产品并不特别要求 PCI 总线，仅对有 PCI 总线图形的计算机进行了测试。 

  2) 对 ISA 总线图形适配器的测试已经通过。然而，在试图恢复到先前的“PM”
     驱动程序时，有时在检测适配器时会出现故障。在恢复安装 VGA 后重新启动
     计算机并初始化适配器后将解决此问题。

     当前不支持 ATI Mach 64 ISA 总线适配器。

  3) 所有的 GRADD 驱动程序，除了 VGAGRADD，都需要支持 VESA** 视频 BIOS ( 版本 1.2 或者
     更高 ) 或者是具有一个适当的 SVGADATA.PMI 文件。对于旧的非 VESA 的但是具有IBM
     的 SVGA PM 驱动程序支持的适配器，可以使用 GENGRADD 作为替代的 SVGA 驱动程序，
     该驱动程序可以通过在 GRADD 图形 BBS 驱动程序 SETUP 安装命令后跟上“SVGA ON INIT”
     以创建一个 SVGADATA.PMI 文件来设置。在使用 GRADD 驱动程序前重新启动。


                OS/2 GRADD 设备驱动程序                                第六页

  4) M64GRADD 和 S3GRADD：发现某些具有硬件光标支持的特定的适配器出现问题。
     故障现象表现为光标显示为一个实心的黑色矩形并且在移过屏幕时会导致显示错误。
     已经提供了一个通融措施来强制使用软件光标。将以下行添加到 CONFIG.SYS 并
     重新启动计算机将可以解决此问题：

        SET HW_CURSOR=OFF

  可以在以下网址找到最新的 GRADD 和图形设备驱动程序的修正和更新：

     http://service5.boulder.ibm.com/pspfixpk.nsf

  在以上 URL 选择“All Fix packages - By Product”然后搜索“GRADD”。


  复制到软盘
  ____________________


  当解压到一个安装目录树后，可以将 GRADD 驱动程序安装文件使用 OS/2 备份命令
  复制到软盘：

  1. 打开 OS/2 全屏幕或者 OS/2 窗口阶段作业。

  2. 输入“BACKUP C:\BBS\GRADD A: /S”并按 Enter 键。

  3. 插入任何已经格式化的磁盘。

  4. 可以使用 HELP BACKUP 来获得关于 BACKUP 命令的更多信息。

  要将此磁盘恢复到相同的或者不同的计算机：

  1. 打开 OS/2 全屏幕或者 OS/2 窗口阶段作业。

  2. 输入“RESTORE A: C:\BBS\GRADD /N /S” 并按 Enter 键。

  3. 插入先前地备份磁盘。

  4. 可以使用 HELP RESTORE 来获得关于 RESTORE 命令的更多信息。

  IBM 未提供可以直接从软盘安装的方法。

  安装
  ____________

  预备步骤
  _________________

  在安装 GRADD 设备驱动程序前通过以下步骤预备好您的计算机：

  1.  应当关闭其它应用程序，特别是 WIN-OS/2 应用程序，以便设备驱动程序的安装过程
      可以替换那些可能被其它应用程序锁定为只读的设备驱动程序文件。

  2.  如果已经安装了一个 SVGA 驱动程序，使用在本自述文件中的
      “把您的显示器设置为 VGA 模式”章节中的指导返回到 VGA 分辨率。


  把您的显示器设置为 VGA 模式
  ________________________________

  如果您的桌面已经可用，我们建议您使用选择安装将您的显示设置为 VGA 模式。
  否则，使用以下过程返回到 VGA 方式。

      1.  打开您的计算机。如果您的计算机已经打开，关闭或者按
          Ctrl+Alt+Del 以重新启动。
      2.  当在屏幕的左上角出现一个小的白色框时，按 Alt+F1 。
      3.  当出现恢复选项屏幕时，按 F3 或者根据操作系统版本按“V”。
      4.  运行选择安装更改 Windows 驱动程序。

  一旦计算机重新引导，您的显示适配器将恢复到 VGA 模式工作。如果您希望安装一个新的
  图形设备驱动程序，参见在“OS/2 Warp 用户指南”中的“视频”章节。


  人工设置的安装
  _____________________

  要进行 GRADD 驱动程序的人工设置的安装，完成以下操作：

  1.  打开 OS/2 全屏幕或者 OS/2 窗口阶段作业。

  2.  在运行解压后的 SETUP.CMD 文件前，使用 SET LANG= 设置正确的语言。
      在 CONFIG.SYS 中已经可能正确的设置了语言。在自述文件中的“解压”
      部分给出了例子。



                OS/2 GRADD 设备驱动程序                                 第八页

  3.  在 OS/2 命令提示下，输入以下命令将所有必须的文件复制到 OS/2 引导驱动器：

          <SrcPath>SETUP xxx  <SrcPath>   <BootDrive>   /u 

      其中：
          xxx 是以下之一：

              “ATI”   - ATI Mach64/Rage Family GRADD
              “CHP”   - Chips & Technology 6555x Family GRADD
              “DYN”   - 动态选定 GRADD
              “GEN”   - Generic VESA Unaccelerated GRADD
              “MGA”   - Matrox Millenium & Mystique Family GRADD
              “NM”    - NeoMagic 2090/2093/2097/2160/2200/2360 Family GRADD
              “S3”    - S3 86x/96x/Trio Family GRADD
              “VGA”   - 视频图示阵列 (VGA) GRADD
              “VIRGE” - S3 Virge/Trio3D Family GRADD

           <SrcPath> SETUP 不在当前目录时，SETUP 的路径。

           <SrcPath>  参数是可选的 ( 如果指定了下一个参数时是必须的 )。

           <BootDrive>  参数是可选的。

           /u  是可选的，表明这是一个自动设置安装，例如 CID 安装。
               不需要 CID 响应文件。

      然后按 Enter 键。

  4.  当提示关机时，关闭并重新启动计算机。

      注意：

      1) 在安装驱动程序期间，将在 OS2\INSTALL 目录中创建和添加 DISPLAY.LOG 和
         DSPINSTL.LOG 文件。这些文件将识别更新的 OS/2 系统文件并表明安装是否
         成功以及失败原因。如果要将安装问题报告给 IBM ，这些文件的内容可能会
         很有用。

      2) 可能会提示说目标文件比源文件更新，是否希望覆盖某些文件。一般说来，
         应当回答“是”，除非您在使用此驱动程序时遇到问题并且正在重新安装。








                OS/2 GRADD设备驱动程序                                第九页

  5.  在计算机重新启动后，完成以下操作正确配置您的显示器并选择一个适当的分解率
      和刷新率：

      a.  打开 OS/2 系统文件夹。
      b.  打开系统设置文件夹。
      c.  打开系统对象。
      d.  当出现设置笔记本时，选择屏幕标签。
          然后转至屏幕的第二页。
      e.  对于支持 VESA** Display Data Channel (DDC) 1 和 2B 协议的情况将自动
          检测 DDC 显示。
          如果您的显示器是 DDC 兼容的，屏幕第二页应当显示您的显示器的一段说明。
          如果您的显示器不兼容 DDC，从显示列表中选择您的显示器。
          如果未列出您的显示器，选择缺省。
      f.  返回到屏幕的第一页并选择分辨率和刷新率。
          根据您的适配器和显示器，可能不能选择刷新率。
      g.  关闭设置笔记本。
      h.  关闭并重新启动您的计算机。
